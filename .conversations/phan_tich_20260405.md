### 4. Phân tích bổ sung nghiệp vụ

#### 4.1 Thi nghiệp vụ — 2 trường hợp

| Trường hợp | Mô tả |
|------------|-------|
| **Vòng loại → Chung kết** | Đăng ký vòng loại, được chọn → vào chung kết |
| **Ghi danh thẳng** | Đăng ký trực tiếp vào chung kết (không qua vòng loại) |

- `competitions`: thêm `has_qualification`, `allow_direct_final`

#### 4.2 Thể thao — Vòng loại, Chung kết, Play-off

| Giai đoạn | Mô tả |
|-----------|-------|
| **Vòng loại** | Các đội đăng ký thi đấu, giành giải để vào danh sách chọn |

#### 4.3 Thi sắc đẹp (Miss) — Chỉ dành cho nữ

## Cập nhật 04/05/2026 — Kiến trúc mới

### 6. Thay đổi kiến trúc hệ thống

**Kiến trúc cũ (React SPA + API):**
- Frontend: React SPA 
- Backend: Yii1 REST API
- Auth: OAuth2 flow với Portal

**Kiến trúc mới (Yii1 Full-stack + External API):**
- Frontend: **Yii1 MVC** (không phải React SPA)
- API: **External endpoint** với API Key cố định
- Auth: **JWT token từ Portal redirect** (không phải OAuth2 flow)

### 6.1 Authentication Flow

```
Portal (portal.muongthanh.vn)
    │
    │ User click "Event Regis" button
    │ → redirect?token=<JWT>
    ▼
Yii1 Frontend (/auth/callback)
    │
    │ Validate JWT → Create Yii Session
    │ - user info
    │ - permissions
    │ - last_activity
    │ - last_refresh
    ▼
Dashboard
```

### 6.2 Session Management

| Config | Value | Mô tả |
|--------|-------|-------|
| `sessionTimeout` | 30 phút | Không hoạt động → session hết hạn → redirect portal |
| `refreshInterval` | 15 phút | Mỗi 15 phút, nếu user vẫn active → gọi Portal refresh token |

### 6.3 API Connection

- API kết nối bằng **API Key** (config trong params.php)
- Không dùng JWT token cho API calls
- API Key đảm bảo kết nối ổn định, không expire

### 6.4 Components đã cập nhật

| Component | File | Mô tả |
|-----------|------|-------|
| `AuthHandler` | protected/components/AuthHandler.php | Decode JWT, quản lý session |
| `ApiClient` | protected/components/ApiClient.php | Gọi External API với API Key |
| `AdminController` | protected/components/AdminController.php | Base controller, check session |
| `PermissionHelper` | protected/components/PermissionHelper.php | Check CRUD permissions |
| `AuthController` | protected/controllers/AuthController.php | Nhận token từ Portal |

### 6.5 Config

```php
// protected/config/params.php
'portalUrl' => 'https://portal.muongthanh.vn',
'portalRefreshUrl' => 'https://portal.muongthanh.vn/api/token/refresh',
'externalApiUrl' => 'https://api.muongthanh.vn/eventregis',
'externalApiKey' => 'YOUR_API_KEY',
'sessionTimeout' => 1800,   // 30 phút
'refreshInterval' => 900,   // 15 phút
```
